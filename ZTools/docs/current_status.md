## 2. 已实现功能

### 2.1 禅道自动化助手 (ZenTao Helper)

✅ **信息收集**
- 查询指派给我的需求列表
  - **默认过滤**：只查询已计划和已立项阶段的需求
  - **查看所有支持**：可通过"所有"/"全部"关键词查看全部需求
  - **未创建任务过滤**：支持过滤出尚未创建任务的需求
  - **标题关键字过滤**：支持按需求标题关键字过滤，关键字可配置
  - **状态显示**：草稿/激活/已关闭/已变更（带 emoji 图标）
  - **优先级显示**：🔴🟡🟢⚪ 图标
  - **阶段显示**：研发中/测试中/已发布等
  - **详细信息**：产品、计划、创建时间、创建者、禅道链接
  - **✅ API 优化**：使用 `my-story-assignedTo-id_desc-{total}-{limit}-{page}.json` API，支持分页查询，最多可查询 2000 条需求
  - **产品过滤**：支持在配置中指定要查询的产品列表，默认只查询"都江堰系统"
  - **关键字配置**：支持在配置中指定默认关键字列表
  - **性能优化**：阶段过滤 -> 关键字过滤 -> 未创建任务检查，减少 API 调用次数
- 查询指派给我的任务列表
  - **默认过滤**：只查询等待和进行中的任务
  - **状态过滤支持**：可通过自然语言指定状态（已完成/进行中/已关闭等）
  - **任务列表显示增强**：创建时间、截止时间、禅道链接、任务状态（带 emoji 图标）
- 按状态分组显示任务（待办/进行中/已完成/已关闭）

✅ **自动化操作**
- 任务拆解：将一个任务拆解为多个子任务
  - **交互式输入**：支持用户交互式输入任务创建信息
    - 需求等级选择（A-/A/A+/A++/B）
    - 需求优先级选择（紧急/非紧急）
    - 需求上线时间选择（下周周一至周五）
    - 任务执行人输入（支持配置默认值）
    - 任务时长输入（根据需求等级自动设置默认值）
    - 任务截至时间选择（根据任务时长智能推荐）
    - 任务标题自定义
  - **需求标题自动更新**：自动更新需求标题
    - 格式：【[需求等级]【标签1】【标签2】【标签N】[需求上线时间]剩余原需求标题
    - 自动提取原标题中的标签（【】括起来的内容）
    - 自动移除原标题中的上线时间
    - 保留原标题的其他内容
  - **配置化任务创建**：支持在配置文件中设置默认值
    - 默认任务执行人
    - 需求等级对应的任务时长
    - 默认需求上线时间规则
    - 默认任务截至时间规则
    - 任务时长阈值（超过则自动调整截至时间）
    - **默认项目名称配置**：支持配置 `default_project_name`，支持部分匹配（如配置"都江堰"可匹配"都江堰项目"）
  - **项目选择**：自动过滤已完成项目，支持交互式选择或自动使用默认项目
  - **需求评审**：变更需求标题后自动评审，将状态从"已变更"改回"已激活"
  - **非交互模式**：支持命令行参数直接传入任务信息，无需交互式输入
- 任务分配：将任务分配给指定用户

✅ **自然语言交互**
- 支持中英文混合指令
- 智能解析任务ID（#123格式）
- 智能解析用户名（@张三 或 给张三格式）
- 支持从自然语言中提取子任务名称

✅ **安全认证**
- 交互式登录，密码不存储
- 会话信息使用系统 keyring 加密存储
- 会话过期自动检测和重新登录

## 8. Trae Skill 使用说明

### 8.1 调用方式
在 Trae 中直接通过自然语言描述需求即可触发 Skill：

```
查看我的禅道任务
拆解任务#123为前端和后端
把任务#456分配给张三
```

### 8.2 Skill 结构
```
.trae/skills/zentao-helper/
├── SKILL.md           # Skill 元数据和文档
├── skill.py           # 主入口（skill_main 函数）
├── requirements.txt   # Python 依赖
├── config/            # 配置文件
└── src/               # 源代码
```

### 8.3 扩展方式
1. 在 `src/nlp/intent_classifier.py` 添加新意图
2. 在 `src/automators/` 创建新处理器
3. 在 `skill.py` 中添加处理逻辑
