# Windows 快捷脚本使用说明

为了方便在 Windows 上使用禅道自动化功能，提供了一个统一的菜单式批处理脚本。

## 主脚本

### zentao-tools.bat - 禅道工具主菜单

一个交互式菜单脚本，集成了所有常用功能。

**使用方法：**
```bash
双击运行 zentao-tools.bat
```

**功能菜单：**
```
============================================
  禅道自动化工具
============================================

  [1] 查看我的需求
  [2] 查看未分配的需求
  [3] 查看我的任务
  [4] 交互式拆解需求

echo  [0] 退出

============================================
```

---

## 功能详解

### 1. 查看我的需求

显示指派给您的需求列表，包含：
- 需求状态和优先级（带 emoji 图标）
- 产品、计划信息
- 创建时间和创建者
- 禅道链接

**默认过滤：**
- 只显示"已计划"和"已立项"阶段的需求
- 只显示"都江堰系统"产品的需求
- 标题包含"特2"关键字的需求

---

### 2. 查看未分配的需求

显示尚未创建任务的需求列表，帮助您发现需要拆解的需求。

---

### 3. 查看我的任务

显示指派给您的任务列表，按状态分组显示：
- 🔵 待办（等待中）
- 🟡 进行中
- 🟢 已完成
- ⚪ 已关闭

---

### 4. 交互式拆解需求

通过交互式菜单完成需求拆解，流程如下：

1. **输入需求ID** - 例如：11754
2. **选择需求等级** - A-/A/A+/A++/B
3. **选择优先级** - 紧急/非紧急
4. **选择上线时间** - 下周周一/周四、下下周周一/周四
5. **输入执行人** - 默认 zhuxu
6. **输入任务时长** - 默认 8 小时
7. **选择截止时间** - 本周周五/下周周五
8. **确认执行** - 显示所有信息，确认后执行

**执行流程：**
```
1. 更新需求标题（添加等级、标签、上线时间）
2. 评审需求（将状态从"已变更"改回"已激活"）
3. 选择项目（使用配置的默认项目或手动选择）
4. 创建任务（在选定项目中创建关联需求的任务）
```

## 首次使用准备

### 1. 确保已安装依赖
```bash
cd .trae/skills/zentao-helper
pip install -r requirements.txt
```

### 2. 配置禅道服务器
编辑 `config/settings.yaml`：
```yaml
zentao:
  base_url: "http://zentao.diansan.com/"
```

### 3. 配置默认项目（可选）
编辑 `config/settings.yaml`：
```yaml
task_creation:
  default_project_name: "研发中心"  # 支持部分匹配
```

### 4. 登录禅道
首次使用任意功能时会提示登录：
```
需要登录禅道...
用户名: [输入您的用户名]
密码: [输入您的密码]
```

登录成功后，会话信息将加密保存在 `.session.enc` 文件中。

## 快捷方式创建

### 创建桌面快捷方式
1. 右键点击 `zentao-tools.bat`
2. 选择"发送到" → "桌面快捷方式"
3. （可选）右键快捷方式 → "属性"
   - 修改图标（选择喜欢的图标）
   - 设置运行方式（最小化/正常窗口）

### 固定到任务栏
1. 右键点击 `zentao-tools.bat`
2. 选择"固定到任务栏"

### 添加到右键菜单（高级）
可以通过注册表将脚本添加到文件夹右键菜单，实现右键快速启动。

## 故障排除

### 问题：提示 "python" 不是内部或外部命令
**解决：**
1. 确保 Python 已安装
2. 将 Python 添加到系统 PATH
3. 或使用 Python 的完整路径修改脚本

### 问题：提示缺少模块
**解决：**
```bash
pip install -r requirements.txt
```

### 问题：会话过期
**解决：**
1. 删除 `.session.enc` 文件
2. 重新运行脚本登录

### 问题：中文显示乱码
**解决：**
脚本已设置 `chcp 65001` (UTF-8)，如果仍乱码：
1. 确保控制台字体支持中文（如"新宋体"、"Consolas"）
2. 在注册表中设置默认代码页为 UTF-8

### 问题：无法连接到禅道服务器
**解决：**
1. 检查网络连接
2. 确认 `config/settings.yaml` 中的 `base_url` 正确
3. 确认禅道服务器可访问

## 自定义配置

### 修改默认查询条件
编辑 `config/settings.yaml`：
```yaml
story_query:
  products:           # 查询的产品列表
    - "都江堰系统"
  keywords:           # 标题关键字过滤
    - "特2"
```

### 修改任务创建默认值
编辑 `config/settings.yaml`：
```yaml
task_creation:
  default_assigned_to: "zhuxu"      # 默认执行人
  default_project_name: "研发中心"   # 默认项目
  grade_hours:                       # 等级对应时长
    "A-": 1
    "A": 2
    "A+": 4
    "A++": 8
    "B": null
```

---

## 注意事项

1. **脚本位置**：`zentao-tools.bat` 需要在 `zentao-helper` 目录下运行
2. **依赖文件**：确保 `skill.py` 和 `config/settings.yaml` 存在
3. **会话文件**：`.session.enc` 保存登录状态，不要删除
4. **配置文件**：修改配置后无需重启，下次运行自动生效
5. **网络要求**：需要能访问禅道服务器
